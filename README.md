## Descripción
Este proyecto es un sistema básico de gestión de restaurante desarrollado en Python utilizando Programación Orientada a Objetos (POO). Permite gestionar productos y clientes mediante clases, objetos y listas.

## Estructura del proyecto

- modelos/producto.py: Clase Producto
- modelos/cliente.py: Clase Cliente
- servicios/restaurante.py: Clase Restaurante que gestiona listas
- main.py: Archivo principal de ejecución

## Tipos de datos utilizados

- str: nombres, correos electrónicos
- int: edades y cantidades
- float: precios de productos
- bool: estado de disponibilidad y cliente frecuente
- list: almacenamiento de objetos

## Conceptos aplicados

- Programación Orientada a Objetos (POO)
- Clases y objetos
- Constructor __init__
- Método __str__
- Listas de objetos
- Modularidad con importaciones

## Reflexión
Este proyecto ayuda a comprender cómo organizar un sistema en Python usando módulos y clases. El uso de identificadores descriptivos y tipos de datos adecuados mejora la claridad y mantenimiento del código.
